# Пульт охраны банка
Этот репозиторий предназначен для внутреннего использования сотрудниками банка "Полтос". Доступ к БД только через учетную запись. Если Вам просто интересна реализация запросов, можете ознакомиться с кодом и свободно его использовать. 

Пульт охраны позволяет подключиться в БД, которая хранит информацию о визитах и карточках сотрудников, и получать из нее необходимую информацию.

### Как установить

Для работы приложения будет необходима установка `Python 3`. После чего нужно подгрузить необходимые пакеты для нормальной работы скрипта с помощью `pip`. Для этого введите в командной строке команду:
```
pip install -r requirements.txt
```
Также необходимо создать переменное окружение с название `.env`, которое содержит данные для доступа к нужной БД. Для успешного доступа необходимо указать следующие параметры:
- `SECRET_KEY` - секретный ключ
- `DATABASE_URL` - данные для доступа к Вашей БД оформленные в виде ссылки по схеме: `postgresql://USER:PASSWORD@HOST:PORT/NAME`.
> **Примечание** - указанная схема релевантна для БД PosgreSQL, если Вы используете другие базы данных, тогда ознакомьтесь с документацией по пакету [dj-database-url](https://github.com/jazzband/dj-database-url?tab=readme-ov-file#url-schema) и выберите там схему по Ваш случай.

### Как пользоваться

С помощью следующей команды в командной строке указываем путь в дерикторию с репозиторием:
```
cd *путь к локальной директории*
```
После этого вводим следующую команду для запуска скрипта:
```
python main.py
```
В результате скрипт выводит в терминал общее кол-во пропусков, кол-во активных пропусков и запрашивает у пользователя id карточки, информацию по которой необходимо получить. После указания id в консоле выведится имя сотрудника, персональный код, дата создания пропуска и статус пропуска - активный или аннулированный.
